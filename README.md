# Cinema_Ticket_Automation
A ticket automation that can be used in movie theaters is coded in this repo. In the project, sql was used in the mssql server environment.



Kullanılan Tablolar
•	Filmler
•	Film Türü
•	Film Yönetmeni
•	Film Yapımcısı
•	Film Başrolleri
•	Müşteri
•	Sinema Salonu
